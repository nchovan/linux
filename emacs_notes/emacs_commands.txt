ORG NOTES:

exit Cx C-c (hold ctrl hit x then c)
new file / open file: C-x C-f (hold ctrl, push x then f)
save file: C-x C-s (hold ctrl, push x then s)
edit org file: emacs ~/Documents/1.org
esc: gets back to typing mode

enable org mode current file: M-x org-mode (hold alt, press x the type org-mode
add to top of file for org mode: MY PROJECT -*- mode: org -*-
